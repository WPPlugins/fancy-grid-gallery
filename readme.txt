Animated Filterable Grid Galleries with Awesome Hover Effects Absolutely FREE!

== Description ==

You can create unlimited grid galleries in seconds, just use this plugin

<h3><a href="http://demos.webcodingplace.com/fancy-grid-gallery-demo/">Demo</a></h3>

Create Galleries from Grid Galleries menu and use shortcode [fgg id="ID_OF_GALLERY"] to display
<br>
You can also merge multiple galleries using this shortcode [fgg ids="COMMA_SEPARATED_IDS"]

<h3>Features</h3>
<ul>
<li>100% Responsive</li>
<li>Grid (1 to 12 Columns)</li>
<li>Animated Filtering</li>
<li>Responsive Light Box</li>
<li>Drag Drop to Sort Images</li>
<li>Import and Export Capability</li>
<li>Categories Support</li>
<li>Hover Effects for Images</li>
<li>Custom Borders</li>
</ul>


== Installation ==

1. Go to plugins in your dashboard and select 'add new'
2. Search for 'Fancy Grid Gallery' and install it
3. Go to Grid Galleries > Add New, and insert images
4. Upload images and fill some additional informations. (Not Necessary)
5. Now paste shortcode of that gallery and Enjoy...
